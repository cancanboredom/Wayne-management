# Wayne 2569 Duty Scheduler — Setup Guide

## Prerequisites

1. **Git** — Download and install from [git-scm.com](https://git-scm.com/downloads)
2. **Node.js** — Download and install from [nodejs.org](https://nodejs.org/) (LTS version recommended)

## First-Time Setup

1. Open a terminal (Command Prompt or PowerShell) and clone the repo:

   ```
   git clone https://github.com/Oat9898/Wayne-management.git
   ```

2. Go into the folder:

   ```
   cd Wayne-management
   ```

3. Double-click **`Preview App.bat`** to start the app.

4. Open your browser and go to **http://localhost:3000**

## Daily Use

Just double-click **`Preview App.bat`** — it will automatically:

1. ✅ Pull the latest code from GitHub
2. ✅ Install any new dependencies
3. ✅ Start the development server

No extra steps needed!
